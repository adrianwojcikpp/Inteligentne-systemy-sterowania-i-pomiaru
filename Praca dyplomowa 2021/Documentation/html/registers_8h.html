◆ FRAME_CODES

enum FRAME_CODES
strong

List of starting frame codes.

Enumerator
SET 

Set register frame. It is used to write a value into a relevant motor control variable.

GET 

Get register frame. It is used to read a value from a relevant motor control variable.

EXE 

Execute command frame. It is used to send a command to the motor control object.

INFO 

Get board info. It is used to retrieve information about the firmware currently running on the microcontroller.

EXE_RAMP 

Exec ramp. It is used to execute a speed ramp.

GET_REVUP 

Get revup data. It is used to retrieve the revup parameters.

SET_REVUP 

Set revup data. It is used to set the revup parameters.

SET_REF 

Set current references. It is used to set the current reference.

◆ STEVAL_ERROR

enum STEVAL_ERROR

List of error codes.

Enumerator
BAD_FRAME_ID 

BAD Frame ID. The Frame ID has not been recognized by the firmware

WRITE_ON_RO 

Write on read-only. The master wants to write on a read-only register

READ_NOT_ALLOWED 

Read not allowed. The value cannot be read.

BAD_TARGET 

Bad target drive. The target motor is not supported by the firmware

OUT_OF_RANGE 

Out of range. The value used in the frame is outside the range expected by the firmware.

BAD_CMD_ID 

Bad command ID. The command ID has not been recognized

OVERRUN_ERROR 

Overrun error. The frame has not been received correctly because the transmission speed is too fast.

TIMEOUT 

Timeout error. The frame has not been received correctly and a timeout occurs. This kind of error usually occurs when the frame is not correct or is not correctly recognized by the firmware.

BAD_CRC 

Bad CRC. The computed CRC is not equal to the received CRC byte.

BAD_TARGET_2 

Bad target drive. The target motor is not supported by the firmware
